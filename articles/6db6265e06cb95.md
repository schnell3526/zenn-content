# クラスタリングとは

似ているもの同士を複数のグループ(クラスタ)に分ける手法である。

# k-means法

## k-means法の概要

1. まずクラスタの中心を表す代表ベクトルを無作為にクラスタの数分だけ決める。

2. 代表ベクトルを固定し、各事例ベクトルに関しては最も近い代表ベクトルのクラスタに所属するとする。

3. 各クラスタに含まれる事例ベクトルの平均を計算し、代表ベクトルの値を更新する。

4. 代表ベクトルの更新の差分を調べ、変化がなくねれば収束したとして終了する。
